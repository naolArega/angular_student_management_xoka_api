## api for xoka angular student manager

### api reference

#### students

to get all students
`method: get `
`http://api.studentangularxoka.io/students`

to get a student
`method: get `
`http://api.studentangularxoka.io/students/[id]`

to create a student
`method: post `
`http://api.studentangularxoka.io/students`

to update a student
`method: patch `
`http://api.studentangularxoka.io/students/[id]`

to delete a student
`method: delete `
`http://api.studentangularxoka.io/students`

and for the rest is the same with a the type only changed
